    # Vue开发相关问题集锦

    # 1.子组件之间切换滚动条问题

    # 问题描述：

    • 当页面切换通过控制子组件显示时，当切换的前页面滚动条移动过显示第二页面内容超过显示滚动条在之前的位置

    • TIP

      1.当滚动条用的是原生的滚动条时，可以通过document.getElementById('test').scrollTop = 0;将滚动条置顶。 2.当使用element-ui的el-scrollbar滚动条，可以设置锚点document.getElementById('test').scrollIntoView(0,0)
